Style Specification (Global Styling Rules)

Typography
• Use a clean, geometric sans-serif typeface family with a high level of legibility.
• Large headers with bold weight; secondary headers slightly lighter.
• Tight line-height in large headings; more spacious line-height for body text.
• Uppercase is used for structural headers and section labels.
• Body text uses regular weight with soft contrast.
• Numeric blocks use oversized numerals, bold weight, and compact spacing.
• Labels appear in rounded containers with minimal padding.

Layout & Spacing
• Layout relies on large white spaces between sections.
• Content blocks are separated by clear margins and structured alignment.
• Vertical rhythm includes consistent spacing increments.
• Grid system emphasizes balance between text and imagery.
• Rounded corners are applied to images, cards, and content containers.

Color & Contrast
• Predominantly white background for all primary sections.
• Dark contrasting footer covering the full width.
• Text uses dark grayscale tones, with subtle use of accent tones for emphasis.
• Interactive elements use solid dark tones with soft hover brightness shifts.
• Light gray background segments used for numerical data or grouped information.

Buttons & Interactive Elements
• Buttons use rounded rectangular shapes with moderate border radius.
• Button text uses bold uppercase typography with high contrast.
• Hover states introduce subtle tonal shift and slight scaling.
• Arrows inside buttons are minimalistic, geometric, and monochromatic.
• Navigation arrows in sliders appear inside circular containers.

Cards & Image Blocks
• Cards use rounded corners with soft shadows or elevated appearance.
• Image blocks scale responsively and maintain edge rounding.
• Cards feature label chips with rounded edges.
• Text inside cards uses clean spacing and consistent alignment.

Reviews Section Styling
• Review cards have soft background tone distinct from page background.
• Rounded corners and subtle drop shadow for depth.
• Icons for quotes are minimalistic and monochromatic.
• Reviewer image appears in a circular frame.
• Reviewer name uses bold typography; role uses lighter weight.

Slider Styling
• Slides maintain consistent padding and spacing between elements.
• Navigation arrows are circular with centered icons.
• Hover interactions introduce subtle scale increase.
• Slider pagination or numbering appears with bold numeric typography.

Footer Styling
• Footer uses full-width dark contrasting background.
• Large bold call-to-action text occupies significant visual space.
• Menu items aligned cleanly in columns with stable spacing.
• Contact information uses consistent typography rules.
• Minimalistic arrow icon for the call-to-action link, high contrast.

Mobile Adaptive Styling
• Typography scales down proportionally while maintaining hierarchy.
• Spacing becomes tighter but still maintains clarity and breathing room.
• Menu becomes full-screen overlay with centered or left-aligned items.
• Cards stack vertically with consistent spacing between them.
• Buttons and interactive elements increase touch-friendly padding.

Imagery
• All images exhibit soft rounding at the corners.
• Imagery is bright, clean, and high contrast for clarity.
• Images extend horizontally to emphasize panoramic effect when used as hero elements.
• Image groups align symmetrically with consistent aspect ratio handling.

General Aesthetic Principles
• Minimalism with strong typography-driven layout.
• Balanced proportions between text blocks and imagery.
• Consistent use of rounding and spacing to unify the interface.
• Smooth transitions for any appearance, hover, or slide motion.
