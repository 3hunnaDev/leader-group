## 2) Визуальная ДНК (обязательно)

- Стиль: clean/minimal/editorial.
- Плотность: много воздуха, без визуальной перегрузки.
- Контраст: очень светлый фон + почти черный текст + мягкий серый для вторичного акцента.
- Формы: крупные скругленные прямоугольники и pill-элементы.
- Декор: минимальный. Никаких тяжелых теней, ярких градиентов и шумных текстур.

## 3) Цветовые токены (фиксированные)

Использовать только эти базовые цвета для UI-слоев и текста:

```css
:root {
  --color-canvas: #efefef; /* фон страницы */
  --color-surface: #f5f5f5; /* секции/карточки */
  --color-surface-2: #e5e6e8; /* светлые метрики */
  --color-surface-dark: #212126; /* тёмные CTA/футер/акцент-карточка */

  --color-text: #24242d; /* основной текст */
  --color-text-soft: #9ea3aa; /* серые слова-акценты в заголовках */
  --color-text-muted: #6d7178; /* вторичный текст */
  --color-text-inverse: #f4f4f4; /* текст на тёмном фоне */

  --color-border-soft: #d9dadd;
  --color-border-strong: #3a3b40;

  --color-white: #ffffff;
}
```

Правила применения:

- Крупные заголовки: `--color-text`, отдельные слова в `--color-text-soft`.
- Основной CTA: тёмный pill (`--color-surface-dark`) + белый текст.
- Футер/контактный блок: только тёмная подложка + светлый текст.

## 4) Типографика

Рекомендуемый стек (максимально близкий к референсу):

```css
font-family: 'Manrope', 'Inter', 'Segoe UI', sans-serif;
```

Правила:

- Заголовки uppercase, очень жирные, плотные по вертикальному ритму.
- Акцент в заголовках делается цветом, не сменой жирности.
- Текст читаемый, без тонких начертаний.

Размеры:

- Hero H1 (desktop): `clamp(56px, 7vw, 96px)`, `font-weight: 800`, `line-height: 0.95-1.0`, `letter-spacing: -0.02em`.
- Section title: `clamp(36px, 4.2vw, 64px)`, `font-weight: 800`, `line-height: 1.0-1.05`.
- Крупные метрики: `72-96px`, `font-weight: 800`.
- Body: `16-20px`, `line-height: 1.35-1.5`, `font-weight: 500`.
- Label/meta: `12-14px`, `font-weight: 600-700`.

## 5) Сетка и отступы

Desktop:

- Контейнер: `1360-1440px`.
- Внешние поля: `32-48px`.
- Сетка: 12 колонок, gutter `24px`.
- Вертикальные интервалы между секциями: `72-120px`.

Mobile:

- 4-колоночная логика.
- Поля: `16-20px`.
- Интервалы секций: `40-64px`.

Шаг отступов (base 8):

- `8, 12, 16, 24, 32, 40, 48, 64, 80, 96`.

## 6) Радиусы, рамки, тени

- Большие медиакарточки: `24-32px`.
- Обычные карточки: `20-24px`.
- Кнопки/chip/pill: `999px`.
- Рамки: `1px`, мягкий серый.
- Тени: очень слабые или отсутствуют.

## 7) Компонентные паттерны

### 7.1 Pill-label

- Небольшой capsule с тонкой рамкой.
- Светлая подложка.
- Текст 12-14px semibold.

### 7.2 Hero-блок

- Двухтонный uppercase-заголовок.
- Описание справа/ниже, max-width `460-560px`.
- Рядом CTA с круглой стрелкой.

### 7.3 Карточки с изображениями

- Крупные скругленные фото.
- Слева сверху chip с названием.
- Справа сверху круглая action-иконка.
- Допустим нижний overlay-блок с описанием.

### 7.4 Блок метрик

- Большинство карточек светлые.
- Одна акцентная карточка тёмная.
- Число — главный визуальный элемент.

### 7.5 Отзывы

- Светлые карточки с цитатой.
- Внизу: аватар, имя, роль.
- Текст 3-5 строк на десктопе.

### 7.6 Тёмный контактный блок

- Большой тёмный slab на всю ширину контейнера.
- Очень крупный CTA-текст (`LET'S TALK` тип логики).
- Тонкая разделительная линия + компактные колонки меню/контактов.

## 8) Стиль изображений

- Только real photo (сельхоз техника, поля, широкие планы).
- Естественный свет, естественная цветокоррекция.
- Без агрессивных фильтров.
- Композиция чистая: горизонт, техника, контекст поля.

## 9) Анимации и интерактив

- Движение сдержанное, функциональное.
- Duration: `180-260ms`, easing: `ease-out`.
- Hover:
  - кнопки: легкое изменение яркости;
  - карточки: максимум `translateY(-2px)`;
  - без пружин, bounce, резких масштабов.

## 10) Mobile-принципы

- Сохранять ту же визуальную лексику, что на desktop.
- Не уменьшать драматургию заголовков.
- Те же pill-кнопки, карточки и радиусы.
- Последовательный вертикальный сторителлинг без тесных двухколоночных зон.

## 11) Чеклист перед merge

- Заголовки uppercase и двухтонные.
- Цвета соответствуют токенам из раздела 3.
- Радиусы консистентны (нет случайных значений).
- CTA-кнопки pill-формы.
- Layout «воздушный», без перегруза.
- В каждом крупном блоке есть чёткая визуальная иерархия (1 доминанта + вторичные элементы).
- Нет визуального шума (сильные тени, кислотные акценты, случайные градиенты).

## 12) Что запрещено

- Добавлять яркие бренд-цвета как основной тон интерфейса.
- Делать мелкие слабые заголовки.
- Использовать сложный декоративный фон.
- Смешивать несовместимые радиусы/анимации/шрифтовые стили.
