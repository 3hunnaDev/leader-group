# ADR Index

ADR (`Architecture Decision Record`) хранит архитектурные решения, чтобы следующий агент понимал не только "что есть", но и "почему так".

## Правила

- Новый ADR добавляется, когда меняется направление проекта, слой ответственности, базовый tooling или важные инженерные ограничения.
- Формат имени: `NNNN-short-kebab-case.md`
- Статусы: `Accepted`, `Superseded`, `Deprecated`, `Proposed`
- Если решение устарело, старый ADR не удаляется, а помечается новым статусом и ссылкой на замену.
