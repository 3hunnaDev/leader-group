# Entities Layer

`entities` хранит доменные сущности и их представление в UI.

Примеры будущих модулей:

- `elevator`
- `project`
- `contact`

Рекомендации:

- сущность включает `model`, `ui`, `api` (по мере необходимости);
- сущности не знают о `pages` и `widgets`;
- экспорт наружу идет через `index.ts`.
